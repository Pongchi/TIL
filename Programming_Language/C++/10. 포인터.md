# 포인터
> 사용할 주소에 이름을 붙인다. 즉, 포인터는 포인터의 이름의 주소를 나타낸다.

간접값 연산자, 간접 참조 연산자 
```c++
#include <iostream>

using namespace std;

int main()
{
    int a = 6;
    int* b;

    b = &a;


    cout << "a의 값 " << a << endl;
    cout << "*b의 값 " << *b << endl;

    cout << "a의 주소 " << &a << endl;
    cout << "*b의 주소" << b << endl;

    *b = *b + 1;
    
    cout << "이제 a의 값은 " << a << endl;  // 7
    return 0;
}
```
